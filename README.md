Анализ исторических данных премии Оскар (1927-2025): исследование тенденций, визуализация и прогнозная модель.

## О проекте

Проект представляет собой комплексный анализ исторических данных о номинациях и победах на премии Оскар. Используя данные с 1927 по 2025 год, проведено исследование ключевых тенденций, анализ самых успешных участников и построение модели машинного обучения для прогнозирования побед.

Основные направления анализа:
* **Exploratory Data Analysis (EDA)**: исследование 100-летней истории премии
* **Статистический анализ**: фильмы, актеры, режиссеры, студии, категории
* **Гендерный анализ**: распределение номинаций и победа по полу
* **Машинное обучение**: прогнозирование победителей с использованием Random Forest

## Ключевые выводы

### Объем и охват данных
* Анализ основан на 11,110 записях за 97 лет (1927-2025)
* В среднем 115 номинаций в год
* Современный период: 120-130 номинаций ежегодно

### Историческая динамика
**Количество номинаций:**
* Рост с 46 в 1929 году до 120-130 в 2020-х
* Наибольшее количество за год: 186 (1943 год)

**Эволюция категорий:**
* Технические категории: рост на 19.1% (с 17% до 36%)
* Творческие категории: снижение на 4.4%
* Специальные категории: снижение на 14.7%
* *Тренд*: Смещение акцента на технические достижения кинематографа

### Конкурентность категорий
**Самые конкурентные категории (низкий процент побед):**
* Музыкальные номинации: 5.0-18.2% побед
* Главные награды (Best Picture): 10.0-16.5% побед  
* Технические категории: 14.4-18.1% побед

**Инсайт:** Даже в самых престижных категориях только 10-18% номинаций заканчиваются победой

### Самые успешные участники
**Топ фильмов по победам:**
1. **Titanic** (1997) - 12 побед
2. **West Side Story** (1961), **Ben-Hur** (1959), **The Lord of the Rings: The Return of the King** (2003) - по 11 побед
3. **Gone with the Wind** (1939), **The English Patient** (1996), **The Last Emperor** (1987), **Gigi** (1958) - по 9 побед

**Актеры с наибольшим количеством побед:**
* **Katharine Hepburn** - 4 победы (абсолютный рекорд)
* **Frances McDormand**, **Meryl Streep**, **Jack Nicholson**, **Ingrid Bergman** - по 3 победы
* **Emma Stone**, **Tom Hanks**, **Elizabeth Taylor**, **Dustin Hoffman** - по 2 победы

**Режиссеры-лидеры:**
* **John Ford** - 4 победы
* **Frank Capra**, **Richard Day**, **William Wyler**, **Dante Ferretti/Francesca Lo Schiavo** - по 3 победы

### Гендерный баланс
* Актерские категории сохраняют баланс номинаций 1:1
* Категории Best Actor и Best Actress обеспечивают равное представительство
* Процент побед у мужчин и женщин сопоставим

### Результаты машинного обучения
* Алгоритм: Random Forest с балансировкой классов (SMOTE)
* Основная метрика: ROC-AUC = 0.61
* Ключевые признаки для прогноза:
  1. Взаимодействие фильма и категории (0.135)
  2. Количество номинаций фильма в текущем году (0.130)
  3. Конкурентность категории (0.124)
  4. Время с последней номинации номинанта (0.122)
  5. Количество предыдущих номинаций номинанта (0.112)
